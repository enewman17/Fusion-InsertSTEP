# Fusion-InsertSTEP

Add-in for Autodesk Fusion to import step files directly into your current design.
<img width="787" height="393" alt="Screenshot 2026-03-23 201320" src="https://github.com/user-attachments/assets/9ce69dba-2461-4140-ab34-2edf61766ff6" />

To install:

1. Download the repo and extract to a folder named InsertSTEP (The folder can be anywhere you'd like to keep it. )
2. Open Fusion and navigate to the Utilites tab > Add-ins.
3. Find the + icon and select "Script or add-in from device"
4. Select the folder InsertSTEP
5. Turn the add-in on by selecting the 'Run' toggle, select the box to 'Run on startup' to enable the add-in everytime Fusion starts.
